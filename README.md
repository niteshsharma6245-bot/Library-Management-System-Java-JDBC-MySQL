# Library Management System

A console-based Library Management System built using Java, JDBC, and MySQL.

## Features

* Add, Update, Delete and Search Books
* Add, Update, Delete and Search Students
* Issue Books
* Return Books
* View Active Issues
* View Overdue Books
* Fine Calculation
* Search Books by Author
* Search Books by Partial Title
* Duplicate Issue Prevention
* Double Return Prevention

## Technologies Used

* Java
* JDBC
* MySQL
* Maven

## Database Tables

* books
* students
* librarians
* issue_records

## Future Enhancements
